# State Reconciliation Contract

When repository state documents disagree:

1. stop implementation
2. identify conflicting files
3. propose canonical wording
4. reconcile docs before new code

No new feature work may proceed while project-control documents materially disagree about:

- thesis
- active step
- product surface
- current capabilities
