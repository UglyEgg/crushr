# Contributing

Thank you for your interest in this project.

## Contribution status

This repository is **not currently accepting external code contributions, pull requests, or unsolicited patches**.

The project is in active, high-churn development under a structured internal workflow. Development is currently performed by the maintainer with assistance from a controlled AI-based development and review process. Until the architecture, workflows, and public contribution standards stabilize, outside contributions are closed.

Please do not submit pull requests at this time.

## What is welcome

External users are still welcome to:

- open issues for reproducible bugs
- report documentation problems
- suggest features or use cases
- ask clarifying questions about intended behavior

Issue reports are preferred over unsolicited code changes while the project remains in this phase.

## Before opening an issue

Please review the relevant project documentation first:

- `AGENTS.md`
- `docs/whitepaper/index.md`
- `docs/foundational_docs/index.md`
- `.ai/STATUS.md`

If your report relates to repository policy, salvage semantics, or review rules, also read:

- `.ai/contracts/README.md`
- `.ai/contracts/QUALITY_GATES.md`

## Documentation boundary

The `docs/` directory is reserved for website-published material.

Internal contracts, planning notes, and project-control material belong under `.ai/` or the repository root, as appropriate.

## Future contribution policy

External contributions may be opened later once the project reaches a more stable phase and the maintainer publishes contribution, review, and acceptance standards for outside collaborators.

Until then, thank you for your interest and for respecting the current development boundary.
